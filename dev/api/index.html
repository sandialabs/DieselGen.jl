API

DieselGen.DieselFuel — Type
DieselFuel(kwh_per_gallon, density_kg_per_l)

Diesel fuel properties used to convert energy to mass and volume.

DieselGen.EfficiencyMap — Type
EfficiencyMap(power_frac, eff)

Power-fraction efficiency curve for a diesel engine.

power_frac and eff must be the same length with strictly increasing fractions in [0, 1].

DieselGen.EngineDesign — Type
EngineDesign(; kwargs...)

Container for diesel engine design parameters.

Fields mirror FASTSim inputs:

  • max_power_kw
  • idle_power_kw
  • ramp_time_s
  • base_mass_kg
  • power_density_kw_per_kg
  • comp_mass_multiplier
  • eff_map
  • fuel

Only keyword arguments are supported when constructing EngineDesign.

Keyword defaults:

  • max_power_kw = 1.0
  • idle_power_kw = 0.013 * max_power_kw
  • ramp_time_s = 6.1
  • base_mass_kg = 61.3
  • power_density_kw_per_kg = 2.13
  • comp_mass_multiplier = 1.4
  • eff_map = default_diesel_map()
  • fuel = default_diesel_fuel()
DieselGen.EngineResult — Type
EngineResult

Container for simulation outputs.

DieselGen.bsfc_g_per_kwh — Method
bsfc_g_per_kwh(design, eff) -> g/kWh

Brake specific fuel consumption from efficiency and fuel properties.

DieselGen.default_diesel_fuel — Method
default_diesel_fuel([T=Float64]) -> DieselFuel

Return typical diesel properties (37.95 kWh/gal, 0.832 kg/L).

Example

fuel = default_diesel_fuel()
fuel_lhv_kj_per_kg(fuel)
DieselGen.default_diesel_map — Method
default_diesel_map([T=Float64]) -> EfficiencyMap

Return a scaled light-duty diesel efficiency map with a 0.30 peak efficiency to match the 1 kW default engine size.

Example

map = default_diesel_map()
eff = efficiency(map, 0.4)
DieselGen.default_hd_diesel_map — Method
default_hd_diesel_map([T=Float64]) -> EfficiencyMap

Return the FASTSim heavy-duty diesel efficiency map (FASTSim CLI defaults).

DieselGen.efficiency — Method
efficiency(map, power_frac) -> eff

Linear interpolation of efficiency at a given power fraction.

DieselGen.efficiency_at_power — Method
efficiency_at_power(design, power_kw) -> eff

Efficiency from the engine map at a given power output.

DieselGen.engine_mass_kg — Method
engine_mass_kg(design) -> kg

Mass scaling from FASTSim: (max_power_kw / power_density_kw_per_kg + base_mass_kg) * comp_mass_multiplier.

Example

design = EngineDesign(
    max_power_kw = 1.0,
    idle_power_kw = 0.013,
    ramp_time_s = 6.1,
    base_mass_kg = 61.3,
    power_density_kw_per_kg = 2.13,
    comp_mass_multiplier = 1.4,
    eff_map = default_diesel_map(),
    fuel = default_diesel_fuel(),
)
engine_mass_kg(design)

DieselGen.simulate_engine — Method
simulate_engine(rpm, torque_nm, dt_s, design; engine_on=nothing) -> EngineResult

Simulate diesel fuel use over a sequence of RPM/torque points.

dt_s may be a scalar or a vector matching rpm.

Example

design = EngineDesign(
    max_power_kw = 1.0,
    idle_power_kw = 0.013,
    ramp_time_s = 6.1,
    base_mass_kg = 61.3,
    power_density_kw_per_kg = 2.13,
    comp_mass_multiplier = 1.4,
    eff_map = default_diesel_map(),
    fuel = default_diesel_fuel(),
)
rpm = [1500, 2000, 2500]
torque = [2.0, 2.5, 1.8]
result = simulate_engine(rpm, torque, 1.0, design)

torque_nm(power_kw, rpm) -> N*m
